# Ecommerce Sales Analysis 📊

Analyzed 10K+ sales records using Python + Pandas to find revenue trends and top products.

### 📈 Key Results
- *Top Product*: Laptop - ₹10,500 revenue
- *Peak Sales Month*: February - ₹4,158 
- *Total Revenue*: ₹27,150

### 🛠️ Tech Stack
Python, Pandas, Matplotlib, Google Colab
